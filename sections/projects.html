Human Activity Recognition (HAR)

Developed a fall detection system with Raspberry Pi and AWS email alerts. The HAR system uses a Joint RGB-Pose-based algorithm to capture spatial and temporal features, enhancing activity recognition accuracy.

  • Data Collection and Preprocessing: Utilizes datasets (HMDB51, UCF50, KTH) and custom data for fall detection, standardizing videos to 640x480 resolution and maintaining key joints (shoulder, knee) in Pose-based videos.
  • Machine Learning Models:
    • ConvLSTM: Recognizes RGB-based activities using spatial and temporal features.
    • LRCN: Combines CNN and LSTM layers for efficient video classification.
    • Pose-based DNN: Uses Mediapipe-extracted pose key points to classify actions.
  • Fall Detection and Notification: Integrates AWS IoT Core and SNS for real-time fall detection and notification via email alerts.

Twitter Sentiment Analysis (TSA)
  • Objective: To classify Twitter data into positive and negative sentiments, helping gauge public opinion on social media.
  • Data Processing: Utilized a dataset of 1.6 million annotated tweets, applying techniques like tokenization, stop word removal, and lemmatization to prepare data for analysis.
  • Models Implemented:
    • Traditional ML Models: Logistic Regression, SVM, Naive Bayes.
    • Deep Learning Models: LSTM for sequential dependencies in text data.
  • Performance Evaluation: Assessed each model based on accuracy, precision, recall, and F1-score, with deep learning models generally showing superior performance.
  • Real-time Inference: Built an interface for live sentiment prediction using a user-friendly tool, integrating machine learning models and an LLM for improved accuracy.
